# Contributing

Thanks for helping with Game Vault.

## Good first contributions

- Docs fixes and screenshots
- UI polish and responsive improvements
- Accessibility improvements
- Template sheet updates
- Small bug fixes in collection, picker, logging, or stats

## Before opening a PR

- Run the local preview from [TESTING.md](/Users/johnseggman/Codex/TESTING.md)
- Verify the app still works with no Google Sheets configured
- Verify the app still works with `Games`, `Plays`, and `Players` CSVs configured

## Scope guidance

Please avoid opening large PRs that:

- introduce a framework rewrite
- add a database/backend requirement to the core PWA
- turn the hosted BGG service into a generic public proxy

Those changes need design discussion first.
